Welcome to invoke-sphinx's documentation!
=========================================

Welcome to invoke-sphinx project !

This project is a collection of invoke tasks, used in the documentation context. It allows to build, clean and publish your documentation, using `Sphinx <http://www.sphinx-doc.org>`_ and the `reStructuredText <http://docutils.sourceforge.net/rst.html>`_ format.

You can find its source code `here <https://github.com/Smile-SA/invoke-sphinx>`_
